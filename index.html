Web Applications

Raincheck Application

This is the very first web application I have deployed online. The inspiration for the app came to me from an outdated system at work. Customer orders were being handled both on paper and digitally leading to a lack of clarity and organization. I felt it would increase productivity and give me a great learning opportunity to create a digital space that managed these orders. I began my work on this poroject on March 19, 2017 and deployed it to heroku on April 3, 2017. This application was a project to both learn the fundamentals of back-end programming and also to create a tool that provided value in a work environment. I now use this application every week to organize and streamline the customer order process at my workplace.

Languages and Frameworks:

-Node.js -MongoDB -Express -Handlebars -HTMl, CSS, Javascript
