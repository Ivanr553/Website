Carla Kunkel: Interior Designer

Description

This is a website that I have created for one of my first clients. I worked with Carla to create the design, layout, and content of the website to bring her business on to the web. This site highlights my goal to create simple and elegant websites that get out of the way of the content. The website consists of three pages: the home page showing her most impressive work, an about me page where visitors can learn about her and contact her, and a gallery page with albums of some of the work she would like to showcase.

Raincheck v2

Description

This is the second version of a web-application that I have made. The goal of this app was to create a tool that would allow the user to organize customer orders that were received and easily add and remove them. Each user has their own account and the application encrypts and authenticates the user upon login and registration. Each user also only has access to customer orders under their account. The back-end of this application is in Node.js and the front end was done using Angular 2.

Languages and Frameworks

Node.js, MongoDB, Express, Angular 2,
Typescript, Javascript, Sass, Html
